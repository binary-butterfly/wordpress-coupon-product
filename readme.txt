== Description ==

Das Plugin stellt ein Gutschein-Produkt bereit, welches sich nach dem Kauf in einen WooCommerce Gutschein umwandelt.
Die Gutscheine werden nach Zahlungs-Abschluss in der Mail von WooCommerce aufgelistet.

Das Plugin ist entstanden, da während der Corona-Pandemie viele Restaurants nicht mehr für Besucher öffnen dürfen.
Ebenso haben viele Restaurants aber eine aktive, sie unterstützende Community, welche ihre Restaurants gerne
durch den Kauf von Gutscheinen unterstützen mag. Da die bestehenden Plugins auf Gift-Cards optimiert waren, passten
diese nicht so sehr, daher hier ein neues, stark auf den Anwendunsfall optimiertes Plugin.
